# Таблица БЖУ

Этот репозиторий содержит структурированную таблицу белков, жиров и углеводов (БЖУ) для различных продуктов.

## О проекте
Цель проекта — упростить доступ к данным о пищевой ценности продуктов и предоставить их в удобном формате (CSV / таблицы), пригодном для:
- анализа питания
- диетических расчётов
- и тд

## Структура данных
В таблице представлены:
- Название продукта
- Белки (г / 100 г) - не готово
- Жиры (г / 100 г) - не готово 
- Углеводы (г / 100 г)

## Использование
Вы можете:
- скачать CSV-файл и использовать локально
- импортировать данные в Excel / Google Sheets
- использовать в своих проектах

## Как внести вклад
Если вы хотите улучшить таблицу:
